# AI Council — твоя персональна Рада AI

PWA-додаток з 5 вкладками: Claude, ChatGPT, Gemini, Perplexity, Council (спільна кімната).

## Що вміє

- 4 окремі чати з кожним AI
- Спільна кімната з 4 режимами:
  - **Паралельно** — питання всім одночасно, бачиш всі відповіді
  - **Синтез Claude** — всі відповідають, потім Claude робить єдиний підсумок
  - **Дебат** — 2 раунди: AI бачать відповіді один одного і переглядають свою позицію
  - **Голосування** — AI ранжує всі відповіді та обирає найкращу
- Вся історія зберігається локально на телефоні
- API ключі — тільки в твоєму браузері, нікуди не надсилаються
- Встановлюється на головний екран як звичайний додаток

## Як запустити на телефоні

PWA потребує HTTPS. Є 3 варіанти (від найшвидшого):

### Варіант 1: GitHub Pages (рекомендую, безкоштовно, 2 хв)

1. Створи новий репозиторій на GitHub, наприклад `ai-council`
2. Завантаж всі 7 файлів у корінь репо (можеш через веб-інтерфейс GitHub: Add file → Upload)
3. Settings → Pages → Source: `main` branch, `/ (root)` → Save
4. Через хвилину отримаєш URL `https://ТВІЙ_НІК.github.io/ai-council/`
5. Відкрий цей URL на Android у Chrome
6. Меню Chrome (⋮) → **Встановити додаток** / **Додати на головний екран**
7. Готово — іконка на робочому столі, запускається як нативна апка

### Варіант 2: На твоєму Synology NAS

У тебе є NAS DS224+, можна хостити прямо на ньому:
1. Web Station → New Website
2. Завантаж файли в папку `/web/ai-council/`
3. Потрібен HTTPS — через reverse proxy Synology з Let's Encrypt, або через Tailscale Funnel
4. Відкрий URL з телефону, встанови як PWA

### Варіант 3: Локальний тест (без встановлення)

Для швидкої перевірки:
```bash
cd ai-council
python3 -m http.server 8080
```
Відкрий `http://localhost:8080` у браузері. PWA features не працюватимуть без HTTPS, але чати — так.

## Перше використання

1. Відкрий апку
2. Натисни шестерню (⚙)
3. Встав свій Anthropic API ключ (`sk-ant-...`)
4. Обери модель — `claude-sonnet-4-5` оптимально для баланс
5. Збережи
6. Пиши питання — заробить

### Коли докупиш інші API:

- **OpenAI**: platform.openai.com → API Keys → створити. Ключ виглядає `sk-proj-...`. Оплата по токенах.
- **Gemini**: aistudio.google.com → Get API Key. Є безкоштовний тір (15 запитів/хв)
- **Perplexity**: perplexity.ai/settings/api → треба депозит $5

Після додавання ключа в налаштуваннях він одразу активується.

## Технічні моменти

- **Нічого серверного** — все в браузері, виклики напряму до API провайдерів через CORS
- **Безпека ключів** — у localStorage. Не синхронізуються. Якщо очистиш дані Chrome → ключі зникнуть.
- **Офлайн** — сам інтерфейс працює, але для запитів потрібен інтернет (очевидно)
- **Мобільна оптимізація** — safe area для iPhone/Android з вирізами, клавіатура не перекриває композер
- **Моделі** — список можна розширити в налаштуваннях (dropdown), пізніше коли вийдуть нові

## Якщо щось не працює

- **"CORS error"** — перевір що URL починається з `https://`. На `http://` (окрім localhost) API не пускатимуть
- **"401 Unauthorized"** — ключ невалідний або прострочений, перегенеруй
- **Claude відповідає помилкою про модель** — список моделей оновлюється, відкрий налаштування і обери іншу зі списку
- **Нічого не зберігається після закриття** — в Chrome у тебе вимкнений localStorage для сайту, увімкни Cookies для цього домену
