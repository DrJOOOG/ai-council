# AI Council v3 — мульти-чат з Радою AI

Повна переробка. Тепер це повноцінний мессенджер-подібний додаток з окремими чатами, рівнями мислення і вибором режиму роботи Ради.

## Головні фічі

**Список чатів + новий чат**
- Кожен чат зі своєю унікальною комбінацією AI
- Назва, учасники, налаштування зберігаються
- Сортується за часом останньої активності

**Рівні мислення (повзунок)**
- Для кожного AI — від найдешевшого/швидкого (сірий) до найрозумнішого/найдорожчого (червоний)
- Claude: Haiku 4.5 → Sonnet 4.5 → Sonnet 4.6 → **Opus 4.7**
- OpenAI: GPT-4o Mini → GPT-4o → o1-mini → o1-preview
- Gemini: Flash 8B → Flash 1.5 → Flash 2.0 → Pro 1.5
- Perplexity: Sonar → Sonar Pro → Sonar Reasoning → Reasoning Pro

**Режими Ради (коли вибрано 2+ AI)**
- **Паралельно** — всі відповідають одночасно, бачиш всі результати
- **Синтез** — Claude узагальнює всі відповіді в одне рішення
- **Дебат** — 1-5 раундів де AI бачать відповіді один одного і відточують позицію
- **Голосування** — Claude ранжує і обирає переможця

**Додатково:**
- 🌐 **Веб-пошук** — Claude може шукати в інтернеті (вбудований tool)
- 🔬 **Research режим** — 3 ітерації: план → аналіз → висновок
- 📎 **Вкладення** — картинки (всі AI), PDF (Claude + Gemini)

**Головне вікно vs повний лог**
- Головне вікно показує тільки ключові відповіді (у Раді — синтез)
- Три крапки → "Повний лог розмови" → видно всі відповіді всіх AI окремо з тегами раундів

**Водяні знаки фону** — фон чату міняється залежно від учасників:
- Чат з Claude → оранжеві A
- Чат з Gemini → сині зірки
- Чат з Perplexity → бірюзові фокуси
- Чат з OpenAI → зелені розетки
- Рада → мозок

**Розрахунок вартості**
- При налаштуванні нового чату збоку показується приблизна ціна одного запиту
- Жовтий при >$0.10, червоний при >$0.50
- Враховує кількість учасників, рівні, раунди дебату, research режим

## Запуск

Зараз апка вже на GitHub Pages за адресою твого репо. Щоб оновити до v3:

1. Скачай zip з цього чату, розпакуй
2. Відкрий `github.com/DrJOOOG/ai-council` у Chrome
3. Видали старі файли: index.html, app.js, sw.js, manifest.json (старі без style.css)
4. Завантаж нові: index.html, app.js, **style.css** (новий!), sw.js, manifest.json
5. Commit changes
6. Через хвилину GitHub Pages оновиться
7. Щоб у встановленій PWA підтягнулась нова версія — відкрий в Chrome звичайному (не з іконки), прокрути вниз потягни щоб оновити 2 рази, або очисти кеш сайту

## Технічні деталі

- Повністю в браузері, без бекенду
- Всі ключі в localStorage — нікуди не надсилаються окрім API провайдерів
- Виклики напряму до Anthropic/OpenAI/Google/Perplexity через CORS
- Вкладення в base64, 8 МБ максимум на файл
- Офлайн кешування через Service Worker (сам інтерфейс, API очевидно потребує інет)

## Обмеження надійного режиму

- **Веб-пошук** працює **тільки для Claude** (має вбудований `web_search_20250305` tool)
- Для OpenAI/Gemini/Perplexity веб-пошуку немає (потребує платні проксі-API)
- **PDF** працює для Claude і Gemini, для інших — попередження
- **Deep Research на години** не реалізований (браузер засинає). Швидкий Research = 3 ітерації, ~2-5 хвилин
